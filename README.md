# Client Spoofer
This mod hides your client name and your mod list from servers.

## Tested Servers
- Cytooxien (`cytooxien.de`)
- HugoSMP (`hugosmp.net`)

## Technical Details
- Changes brand name
- Fixes translation exploit (sign & anvil)
- Hides known packs
- Blocks custom payload packets
